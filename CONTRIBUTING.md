# Guía de Contribución

¡Gracias por tu interés en contribuir a PC Futbol Remake! Tu ayuda es fundamental para hacer de este un gran proyecto.

## Cómo Contribuir

-   **Reportando Bugs**: Si encuentras un error, por favor, crea un [Issue](https://github.com/mapiedra/pcfutbol-remake/issues) y descríbelo con el mayor detalle posible, incluyendo pasos para reproducirlo.
-   **Sugiriendo Mejoras**: ¿Tienes una idea para una nueva característica o una mejora? ¡Nos encantaría escucharla! Crea un [Issue](https://github.com/mapiedra/pcfutbol-remake/issues) o únete a las [Discussions](https://github.com/mapiedra/pcfutbol-remake/discussions).
-   **Escribiendo Código**: Si quieres contribuir con código, sigue estos pasos:
    1.  Haz un fork del repositorio.
    2.  Crea una nueva rama para tu característica (`git checkout -b nueva-caracteristica`).
    3.  Haz tus cambios y asegúrate de que el código sigue el estilo del proyecto.
    4.  Escribe pruebas para tus cambios si es posible.
    5.  Envía un Pull Request.

## Estilo de Código

-   Usamos ESLint para mantener un estilo de código consistente. Por favor, ejecuta `npm run lint` antes de enviar tus cambios para asegurarte de que todo está en orden.
-   Sigue las convenciones de nombres y la estructura de archivos existentes.

## Proceso de Revisión

-   Todos los Pull Requests serán revisados por los mantenedores del proyecto.
-   Es posible que se te pidan cambios antes de que tu Pull Request sea aceptado.

Al contribuir a este proyecto, aceptas que tus contribuciones se licencien bajo la misma licencia [CC BY-NC-SA 4.0](LICENSE) que el resto del proyecto.
